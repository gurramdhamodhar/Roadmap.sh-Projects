Farmer Welfare System using ML and DL

This project utilizes advanced data analytics and artificial intelligence, to enhance crop management and yield. It integrates crop prediction, disease detection, and fertilizer recommendation. Crop prediction utilizes historical and real-time data, while disease detection employs image recognition and deep learning to identify early signs of diseases, thereby promoting sustainable farming practices.

Airline Sentimental Analysis using NLP

This project aims to analyze the sentiment of flight experiences shared by passengers using natural language processing (NLP) techniques. By applying sentiment analysis to textual data from customer reviews and feedback, the project seeks to gain insights into the positive and negative aspects of the flight experience, enabling airlines to improve their services based on customer feedback.
